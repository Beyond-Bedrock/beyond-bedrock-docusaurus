# Statistics Add-on

Statistics Supplement. It is an add-on that you can use to see your stats and progress in the world or on a server. It is divided into Statistics and Top.

## Features Statistics:

- **Mobs Kills:** This feature enables you to see how many mobs you have killed. Click on the Mobs button to show you each mob and how many times it has been killed:

![Mobs Kills Statistics](/img/addons/statistics-add-on/mobs-kills-statistics.webp)

- **Players Kills:** This feature enables you to see how many player you have killed.
- **Steps:** This feature enables you to see the number of steps/blocks walked in this world.
- **Timer:** Using this feature, you can see the time you have spent in this world since your first entry.
- **Death:** This feature allows you to see how many times you have died.
- **Blocks Break:** This feature allows you to see the total number of blocks that have been broken.

## Features Top:

Using the statistics top, you can see the top 3 players in the world by each statistics. You can see the statistics of the top 3 players in the statistics in which they occupied the first three places:

![Top Players Statistics](/img/addons/statistics-add-on/top-players-statistics.webp)

## How to Use:

- Make sure the **resource pack** is installed:

![Resource Pack Installation](/img/addons/statistics-add-on/resource-pack-installation.webp)

- Make sure the **behavior pack** is installed:

![Behavior Pack Installation](/img/addons/statistics-add-on/behavior-pack-installation.webp)

- Make sure that the **Beta APIS** is activated (if you don't activate it, the add-on will not work):

![Beta APIs Activation](/img/addons/statistics-add-on/beta-apis-activation.webp)

- Write in the chat **!Menu** and exit the chat quickly:

![Chat Command](/img/addons/statistics-add-on/chat-command.webp)
